## About the Role

We're seeking an experienced Senior Unreal Engine Developer to join our team and help create an innovative social gaming platform. As a key member of our development team, you'll be responsible for architecting and implementing core features of our virtual world.

## Responsibilities

- Lead the technical development of key game features using Unreal Engine 5
- Design and implement scalable multiplayer systems
- Create robust character customization and animation systems
- Optimize performance for various platforms
- Mentor junior developers and contribute to technical decision-making
- Collaborate with artists and designers to implement game features
- Participate in code reviews and maintain high code quality standards

## Required Skills & Experience

- 5+ years of professional game development experience
- 3+ years of experience with Unreal Engine (UE4/UE5)
- Strong C++ programming skills
- Extensive experience with multiplayer game development
- Proven track record of shipping games or interactive experiences
- Experience with version control systems (Git)
- Strong understanding of game optimization techniques
- Excellent problem-solving and debugging skills

## Preferred Qualifications

- Experience with social features and real-time communication systems
- Knowledge of character customization systems
- Background in developing live service games
- Experience with cloud services (AWS, Azure, etc.)
- Understanding of CI/CD pipelines
- Experience with Agile development methodologies

## Current Projects

You'll be working on exciting features including:

- Advanced character customization system
- Real-time social interaction features
- Custom animation systems
- Performance optimization for multiple platforms
- Scalable multiplayer infrastructure
